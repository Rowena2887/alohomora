Computers and Software (CS 101)

Covers the organization and operations of a computer and computer networks, information representation, the primary tasks of operating systems, application and system software, and the fundamentals of computer programming. Includes logical reasoning, and managing complexity.

Computer Programming Fundamentals I/II (CS 140/145)

Teaches the fundamentals of computer programming. Covers computer architecture, machine instruction processing, basic data types, program control structures, functional decomposition, classes, and fundamental data structures.

Data Structure and Algorithm Fundamentals (CS 240)

Teaches software development skills that emphasize the study of abstract data types using object oriented programming techniques, Big O algorithm analysis, fundamental data structures such as lists, stacks, queues, and trees; and searching and sorting.

Computer Literacy (CIS 100)

Introduction to the practical application of computers and communication technology. Includes the major components of computer systems, user interfaces, applications, networking, and societal issues surrounding computing, including ethics, privacy, security, and sustainability

Open Source Operating Systems (CIS 106)

Fundamental management of open source systems from the command line, user administration, file permissions, software configuration and management of clients.

Introduction to Computer Security (CIS 110)

Basics of Computer Security, including identifying threats, planning for business continuity, and preparing for various security attacks.

Virtualization (CIS 116)

Implementing virtualization techniques and technologies.

Computer Operating Systems I/II (CIS 105/205)

Introduces the fundamentals of computer operating systems including history, evolution and design, as well as support, maintenance and troubleshooting. Advanced study of computer operating systems and platforms.

Network Security I/II (CIS 214/215)

Network security fundamentals including defining a security policy, attack methods, intrusion detection, firewalls, identifying risks, and securing networks with added emphasis on defense in depth.

Industrial Control Systems Security (CIS 216)

Securing Industrial Control Systems including identifying risks, configuring devices and protocols, attack methods, and security ICS networks.

Cisco Networking I/II/III/IV (CIS 226/227/228/229)

This course introduces the fundamentals of networking, including introduction to the OSI and TCP/IP network models, and IP addressing and sub-netting.
Other topics include:
• Network design, topologies, protocols, wiring, and network devices.
• Routing protocols and concepts including static and dynamic routing. Students will gain hands on experience in the lab configuring and troubleshooting multiple routing protocols, including OSPF.
• LAN Switching and wireless communication, configuring, verifying, and troubleshooting VLANs, inter-VLAN routing, VTP, and trunking on Cisco switches.
• Voice and video over IP and WAN technologies including PPP, Frame Relay, and broadband links. WAN security concepts are discussed in detail, including types of threats, how to analyze network vulnerabilities, and general methods for mitigating common security threats.
